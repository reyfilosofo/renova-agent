# NOUS Intelligence Rooms

**OpenAI Build Week 2026 submission package**  
**Creator:** Carlos Jonathan González Rodríguez · NOUS by SERESARTE

NOUS Intelligence Rooms transforms fragmented organizational information into an evidence-backed decision architecture: executive synthesis, strategic signals, prioritized decisions, a 30/60/90-day roadmap and a traceable evidence ledger.

## Run locally

```bash
./run.sh
```

Open `http://127.0.0.1:8000`.

The app works immediately in curated demo mode. Live GPT-5.6 analysis is enabled when `OPENAI_API_KEY` is available in the server environment. The OpenAI Responses API request uses `store: false`, Structured Outputs, multimodal file inputs and a configurable model via `OPENAI_MODEL`.
