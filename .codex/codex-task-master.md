# Codex Master Task — RENOVA Agent

Act as a senior full-stack engineer and product architect.

## Goal

Bring this repository to a deployable, reviewable and professionally documented first version.

## Required sequence

1. Read `README.md`, `AGENTS.md`, `docs/RENOVA_AGENT_SPEC.md`, `docs/ARCHITECTURE.md`.
2. Run `npm install` if dependencies are absent.
3. Run `npm run check:structure`.
4. Run `npm run build`.
5. Fix build or typing issues.
6. Improve accessibility if needed.
7. Produce a final report.

## Output report

Return:

- Files changed
- Commands run
- Build status
- Remaining risks
- Pull request title
- Pull request body
