# Productos del Laboratorio

## 1. NOUS Recursive Intelligence Sprint

Diagnóstico estratégico intensivo para convertir información dispersa en criterio, decisión y ruta de acción.

### Cliente ideal

Fundadores, artistas, instituciones culturales, directivos, consultores, proyectos editoriales y organizaciones que necesitan claridad estratégica rápida.

### Entregables

- mapa de situación
- diagnóstico de contradicciones
- matriz de prioridades
- ruta de 30 días
- narrativa institucional
- oportunidades de producto
- riesgos principales
- recomendaciones ejecutivas

## 2. ℛenova Self-Renewal Audit

Auditoría de capacidad renovativa para proyectos, marcas, instituciones o corpus editoriales.

### Evalúa

- propósito
- identidad
- narrativa
- archivo
- decisiones
- productos
- criterios de corrección
- consistencia pública
- capacidad de aprendizaje

### Entregables

- Índice ℛ de Capacidad Renovativa
- reporte de hallazgos
- mapa de bloqueos
- plan de renovación
- checklist de continuidad

## 3. SERESARTE Cultural Intelligence Lab

Laboratorio cultural para convertir programas artísticos, educativos o comunitarios en sistemas de aprendizaje, publicación y memoria.

### Entregables

- diagnóstico territorial o institucional
- diseño de laboratorio
- bitácora comunitaria
- reporte de impacto
- publicación final
- exposición o activación
- archivo de memoria

## 4. LUCEM Verification Protocol

Servicio de verificación epistemológica, documental y estratégica.

### Revisa

- fuentes
- citas
- afirmaciones
- contradicciones
- sesgos
- consistencia conceptual
- riesgos reputacionales
- límites de evidencia

## 5. ℛenova Press Intelligence Series

Serie editorial para publicar los hallazgos del laboratorio.

### Formatos

- manifiestos
- white papers
- libros breves
- reportes estratégicos
- cuadernos metodológicos
- casos de estudio
- manuales internos

## 6. ℛenova Agent / NOUS Agent Prompt Pack

Paquete de asistentes especializados para investigación, estrategia, verificación y edición.

### Asistentes

- ℛenova Agent
- NOUS Strategy Agent
- LUCEM Verification Agent
- SERESARTE Editorial Agent
