# Metodología RSR: Recursive Self-Renewal

## Definición

**RSR / Recursive Self-Renewal** es una metodología para que un sistema humano, cultural, editorial, institucional o estratégico observe sus propios patrones, detecte errores, conserve memoria, verifique hipótesis, modifique criterios y produzca mejores decisiones.

No es automatización total.
No es autoayuda.
No es AGI.
No es una promesa de autonomía sin supervisión.

Es un sistema de mejora asistida, versionada y verificable.

## Ciclo RSR

### 1. Observar

Recolectar documentos, decisiones, discursos, productos, piezas editoriales, errores, métricas, intuiciones y conflictos.

### 2. Diagnosticar

Identificar patrones, contradicciones, bloqueos, fuerzas, vacíos, oportunidades y riesgos.

### 3. Archivar

Guardar versiones, criterios, decisiones, hipótesis, descartes y aprendizajes.

### 4. Verificar

Aplicar LUCEM: revisar fuentes, claims, consistencia, evidencia, límites y trazabilidad.

### 5. Renovar

Modificar el criterio, rediseñar el producto, ajustar la narrativa, crear el siguiente documento, corregir el sistema.

### 6. Publicar

Transformar hallazgos en entregables: reporte, white paper, landing, libro, manifiesto, guía, presentación o tablero.

### 7. Reingresar

El nuevo producto entra al archivo y se convierte en insumo del siguiente ciclo.

## Fórmula operativa

```txt
RSR = Observación + Diagnóstico + Archivo + Verificación + Renovación + Publicación + Reingreso
```

## Indicadores iniciales

- claridad de propósito
- consistencia narrativa
- trazabilidad documental
- velocidad de corrección
- calidad de fuentes
- reducción de contradicciones
- aprendizaje entre versiones
- capacidad de publicación
- utilidad estratégica
- continuidad institucional
