# Renova Lab Kit

Renova Lab Kit is the applied facilitation layer of Renova.

It is designed for schools, cultural centers, teams, civic labs, reading groups, workshops, and institutional reflection sessions.

## Basic session structure

1. Habitat: identify enabling and obstructive conditions.
2. Fracture: name the relevant rupture or overload with precision.
3. Horizon: define a credible future within a specific time frame.
4. Care practice: convert the diagnosis into a concrete action.

## CLI usage

```bash
python -m renova_core.cli canvas "School renewal workshop"
```

## Output

The command renders a Markdown facilitation canvas that can be copied into a workshop note, printed, or adapted for Obsidian.

## Real-world uses

- Cultural project diagnosis.
- Community workshops.
- Educational reflection sessions.
- Organizational alignment.
- Public program design.
- Documentation of learning circles.
