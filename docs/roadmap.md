# Roadmap

## v0.1

- Escritorio web funcional.
- Boot screen.
- Ventanas movibles.
- Terminal con comandos basicos.
- Sistema de archivos virtual.
- Apps: Terminal, Archivos, Notas, Navegador interno, Calculadora y Sistema.
- Persistencia en `localStorage`.

## v0.2

- Historial persistente de terminal.
- Autocompletado de rutas y comandos.
- Mejoras de accesibilidad para mover ventanas con teclado.
- Busqueda dentro del explorador de archivos.
- Vista previa Markdown en Notas.
- Pruebas automatizadas de navegador.

## v0.3

- Importacion/exportacion ZIP del FS virtual.
- Papelera virtual.
- Multiples notas abiertas.
- Temas visuales configurables.
- Manifest PWA opcional.

## v0.4

- Tablero de proyectos RENOVA/SERESARTE.
- Bitacora de sesiones.
- Biblioteca local simulada.
- Integracion opcional con backend real.

## v0.5

- Modo agente controlado.
- Cola de tareas.
- Auditoria de acciones.
- Separacion entre memoria local, archivos virtuales y conectores externos.
