# Renova Open Corpus

Renova Open Corpus is the canonical public knowledge layer of the project.

It should contain definitions, axioms, glossary entries, public essays, translation notes, bibliographic records, and machine-readable data that can be reused by the agent, the index engine, the graph, and the lab kit.

## Current files

- `data/glossary_min.json`: initial public glossary.
- `renova_core/corpus.py`: utilities for loading and searching glossary files.
- `docs/*.md`: public documentation for the open-source system.

## Future files

- `CITATION.cff` for scholarly citation.
- `bibliography.bib` for academic references.
- multilingual folders for Spanish, English, German, French, and Italian.
- canonical definitions with version numbers.

## Governance rule

The open corpus should be stable enough to cite, but flexible enough to improve through versioning.
