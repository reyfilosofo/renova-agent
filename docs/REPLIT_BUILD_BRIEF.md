# Replit Build Brief

Build Renova Open System as a functional app.

## Objective

Create an interactive tool for Renova. The app should help users assess renewal capacity, explore concepts, ask a guided agent, and generate workshop canvases.

## Screens

1. Dashboard
2. Index assessment
3. Agent
4. Concept explorer
5. Glossary
6. Lab canvas
7. Methodology

## Behaviors

- Calculate IRG from five scores.
- Show a qualitative level.
- Accept evidence notes.
- Render a report.
- Let the user ask the guided agent a question.
- Show concept nodes and relations.
- Search glossary terms.
- Generate a workshop canvas from a subject.
- Work on mobile and desktop.

## Style

Serious, precise, philosophical, institutional, and clean.

## First build

Use local data and local state. Do not require external APIs, accounts, payments, or a database for the first version.

## Later roadmap

- saved assessments
- export to PDF
- multilingual mode
- GitHub sync
- model-backed agent
- link or embed from existing Wix presence
