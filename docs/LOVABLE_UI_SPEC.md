# Lovable UI Specification

## Goal

Create a polished app prototype for Renova Open System.

This should be treated as a functional web app prototype, not as a promotional website.

## Visual identity

- serious philosophical technology
- institutional clarity
- deep dark background
- ivory content cards
- restrained gold accent
- clean typography
- calm navigation
- high readability

## Screens

1. App dashboard
2. Renova Index
3. Renova Agent
4. Renovagrama
5. Open Corpus
6. Lab Kit
7. Methodology

## Dashboard cards

Each module card should show:

- title
- short description
- current status
- primary action

## Interaction requirements

- navigation sidebar
- responsive layout
- assessment form
- score result panel
- agent question panel
- glossary search panel
- concept explorer panel
- canvas generator panel

## Copy direction

Renova is presented as open cultural technology for life-oriented systems thinking, renewal diagnostics, philosophical education, and practical facilitation.
