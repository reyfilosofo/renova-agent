# Wix Link Plan

## Decision

The Renova project should remain an app in Replit for this phase.

## Wix role

Wix can later support the app as:

- discovery surface
- link from the existing Renova presence
- contact layer
- booking layer
- newsletter layer
- payment or membership layer if needed later

## Current account context

The account already has Renova-related Wix sites. They should point to the Replit app after the app URL exists.

## Next step after Replit preview exists

1. Add a button from the Renova Wix presence to the Replit app.
2. Optionally embed the Replit app preview if allowed.
3. Keep app logic in Replit.

## Rule

Replit is the executable app. Wix is distribution and institutional presence.
