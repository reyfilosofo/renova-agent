# Notas de licencia y uso

Este repositorio separa dos capas:

1. Código fuente.
2. Obra cultural, marcas, nombres, textos, imágenes, manifiestos y símbolos.

## Código

El código puede publicarse bajo una licencia open source si SERESARTE lo decide. Una opción común para starters técnicos es MIT, siempre que se revise antes de producción.

## Obra cultural

Los nombres SERESARTE, ℛenova, Rey Filósofo, Renova Press, NOUS y sus símbolos asociados deben tratarse como identidad cultural y de marca. La posesión de un NFT, badge, certificado o pass no debe interpretarse como cesión automática de derechos de explotación comercial.

## Cláusula operativa sugerida

La recepción o compra de un token asociado a SERESARTE / ℛenova permite conservar, visualizar o acreditar la pieza digital según las condiciones específicas del proyecto. No concede, por sí sola, derechos de reproducción comercial, sublicencia, modificación, registro de marca o uso institucional de la obra.

## Privacidad

No publicar en blockchain información sensible ni archivos internos. Para documentos privados, registrar hashes y metadatos mínimos, no el contenido completo.

## Advertencia

Antes de vender, desplegar o usar públicamente este sistema, revisar contratos, licencias, condiciones de uso, impuestos y privacidad con profesionales competentes.
