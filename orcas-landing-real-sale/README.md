# El secreto de las orcas - landing real de venta

Landing site estático para vender el PDF digital **El secreto de las orcas** de SERESARTE / Renova Press.

## Estado

- Rama creada: `orcas-landing-real-sale`
- Paquete local completo generado: `ORCAS_EDGEONE_REAL_VENTA_9MB_PLUS.zip`
- Paquete directo para EdgeOne generado: `ORCAS_EDGEONE_DEPLOY_REAL_SITE_ONLY.zip`
- Contiene imágenes reales extraídas del PDF final, preview con marca de agua reforzada y descarga cifrada por códigos ORCA.

## Instrucciones

1. Descargar el paquete EdgeOne generado en ChatGPT.
2. Editar `site-config.js` con WhatsApp, correo, Payhip y Ko-fi reales.
3. Subir el contenido del ZIP `ORCAS_EDGEONE_DEPLOY_REAL_SITE_ONLY.zip` a EdgeOne Pages Drop.
4. Mantener `access-codes/ORCAS_CODES_100.csv` fuera del sitio público.

## Nota

No se sube el PDF cifrado ni los JPG pesados directamente al repositorio desde este commit para evitar saturar el conector de GitHub. El paquete completo está generado como artefacto descargable.
