# AGENTS.md

This directory is a Markdown knowledge vault for SERESARTE, Renova, NOUS and DAO projects.

## Rules

- Work in Markdown.
- Preserve existing files.
- Create backups before large edits.
- Use Obsidian-style links when helpful.
- Record important changes in `09_META/CHANGELOG.md`.
- Mark uncertain information as `requires verification`.

## Workflow

1. Read `README.md` and `MAPA_MAESTRO.md` first.
2. Classify new notes into the correct folder.
3. Add front matter to important notes.
4. Create indexes and cross-links.
5. Summarize changes after each task.
