# 00_INBOX

Carpeta de entrada para notas rápidas, capturas, transcripciones, borradores sin clasificar y materiales recibidos.

## Uso

1. Depositar aquí todo lo que todavía no tiene categoría.
2. Revisar periódicamente.
3. Mover cada elemento a su carpeta definitiva.
4. Registrar decisiones relevantes.

## Estados sugeridos

- sin revisar
- clasificar
- convertir a Markdown
- archivar
