# Roadmap SERESARTE Web Lab

## Fase 1: Base operativa

Estado: iniciado.

Entregables:

- Carpeta `seresarte-web-lab`.
- Dashboard estatico.
- Registro `data/projects.json`.
- Guia de despliegue.
- Gobernanza editorial y tecnica.

## Fase 2: Landings prioritarias

Objetivo: convertir el ecosistema SERESARTE en una familia de sitios claros y publicables.

Prioridad inicial:

1. Fundacion SERESARTE.
2. SERESARTE Experiences.
3. SERESARTE Media.
4. Renova Press.
5. ReyFilosofo x SERESARTE.

## Fase 3: Experiencias comerciales

Objetivo: crear paginas vendibles y accionables.

Proyectos:

- Entra al Lienzo.
- Mural Colectivo.
- Voluntariado corporativo.
- RSE/RCE.
- Natura Movil.
- Arte con causa.

## Fase 4: SERESARTE HTML

Objetivo: crear coleccion de piezas interactivas comercializables.

Entregables:

- Plantilla de pieza.
- Metadata por obra.
- Preview.
- Precio.
- Boton de contacto.
- Marca editorial.

## Fase 5: Juegos y educacion

Objetivo: prototipar juegos culturales, visuales y educativos.

Proyectos posibles:

- Juego de memoria de colores.
- Juego de tipos de nubes.
- Juego de animales.
- Juego de trazos y pintura.
- Juego de preguntas filosoficas.

## Fase 6: Automatizacion

Objetivo: desplegar y validar automaticamente.

Automatizaciones posibles:

- Build checks.
- Validacion de HTML.
- Revision de enlaces.
- Compresion de imagenes.
- Deploy previews.
- Lighthouse basico.
