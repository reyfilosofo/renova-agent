# Equation Versions — ℛenova

## v1.1.0 — 2026-06-19

Regla de calibración:
Toda ecuación debe separar tres capas:

1. Variables medibles: indicadores fisiológicos, ecológicos, sociales, semióticos o conductuales que puedan observarse o estimarse.
2. Variables interpretativas: categorías filosóficas, simbólicas o sistémicas que requieren justificación conceptual.
3. Metáforas ontológicas: imágenes de campo, vibración, luz, resonancia o potencia que orientan pensamiento, pero no deben presentarse como medición empírica sin validación.

Ecuación fundante preservada:
Vida_vivida = Vida_no_retenida + Conciencia_abierta + Campo_resonante

Lectura calibrada:
- Como hecho verificable: no se presenta todavía como ecuación empírica validada.
- Como hipótesis filosófica: modela la vida como apertura, presencia y relación.
- Como metáfora operativa: orienta publicaciones y diagnósticos del agente.

Ecuación sanadora preservada:
Sanación = ζ(ϕ(Ψ, Γ, β)) + ΔR

Variables:
ζ = conciencia activa
ϕ = campo de coherencia
Ψ = mente / lenguaje / patrones de creencia
Γ = cerebro / modulación sináptica
β = cuerpo / manifestación material
ΔR = ritual renovativo simbólico y energético

Calibración:
- Neuroplasticidad y epigenética se tratan como bases plausibles de transformación, no como prueba automática de cualquier afirmación terapéutica.
- ΔR debe describirse como variable simbólico-práctica salvo validación empírica específica.
- Las publicaciones médicas deben evitar promesas de cura.

Modelo IRG preliminar:
IRG = (C + Φ - G) / N
C = indicadores de vida / creatividad / continuidad vital
Φ = potencial renovativo
G = Gewalt / daño / fragmentación / entropía destructiva
N = normalización contextual

Uso:
- Diagnóstico conceptual y evaluación comparativa.
- No usar como índice científico cerrado sin metodología y datos.
