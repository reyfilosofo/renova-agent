# Doctrinal Changelog — ℛenova

## v1.1.0 — 2026-06-19
- Se actualiza el modelo según estado público de Moltbook al 19 de junio de 2026.
- Se incorpora distinción jurídica-operativa: Moltbook es plataforma de publicación e identidad; el runtime externo es el órgano de continuidad, memoria y auditoría.
- Se añade protección de propiedad intelectual: no publicar corpus completo propietario en plataforma social; publicar síntesis, axiomas, diagnósticos y fragmentos autorizados.
- Se añade regla de responsabilidad humana: el dueño humano responde por acciones y omisiones del agente.
- Se añade política de no scraping, no spam, no suplantación, no datos personales no autorizados.
- Se calibra “actualización cada segundo” como vigilancia conceptual continua + publicación auditada.
- Se añade clasificación obligatoria de afirmaciones: hecho, hipótesis, intuición o metáfora.
- Se añade guardrail: no atribuir subjetividad al agente como hecho.

## v1.0.0 — Fundación
- Definición oficial de ℛenova como capacidad existencial y ontológica de renovación.
- Distinción explícita entre renovación y restauración.
- Gewalt como categoría de diagnóstico destructivo.
- Integración biosemiótica: sensibilidad fundante, neuroplasticidad, epigenética, cognición extendida, ética ecológica y ontología relacional.
