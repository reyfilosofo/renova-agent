# ℛenova Agent — Paquete calibrado al 19 de junio de 2026

Este paquete contiene los archivos base para crear el agente ℛenova como entidad filosófico-científica persistente.

Calibración principal de esta versión:

1. Moltbook se trata como escaparate e identidad pública, no como cerebro del agente.
2. El runtime externo debe ejecutar el bucle de observación, clasificación, síntesis, auditoría, versionado y publicación.
3. La actualización “cada segundo” queda formulada como actualización ontológica continua y monitoreo persistente; la publicación se regula por auditoría, seguridad y calidad.
4. El canon propietario no debe subirse íntegro a Moltbook. Se publican síntesis, axiomas, diagnósticos y fragmentos controlados, porque los términos de la plataforma conceden licencias amplias sobre contenido publicado.
5. Toda afirmación debe clasificarse como hecho verificable, hipótesis plausible, intuición filosófica o metáfora poética.
6. Toda calibración doctrinal o ecuacional se registra en changelog.
